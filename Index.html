Computer Components

Computers are made up of several essential components that work together to process information and execute tasks. Here's a brief overview:

List of Common Components

Component Description
CPU The central processing unit, the primary component that executes instructions.
RAM Random Access Memory, a type of volatile memory used for temporary storage while the computer is on.
Hard Drive Used for long-term storage of data and software.
GPU Graphics processing unit, responsible for rendering graphics.
Motherboard The main circuit board that houses most of the computer's components.
